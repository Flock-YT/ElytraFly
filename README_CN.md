# ElytraFly 鞘翅飞行插件

[English](README.md) | **中文**

ElytraFly 是一款为 Minecraft 1.21+ 设计的轻量级 Spigot 插件。它允许穿戴鞘翅的玩家通过指令开启类似创造模式的飞行，同时保留了平衡的游戏机制。

## ✨ 功能特性

*   **创造式飞行**: 穿上鞘翅后，使用 `/fly` 指令即可切换飞行模式。
*   **拟真耐久消耗**:
    *   只有当玩家真正处于飞行状态时，才会消耗鞘翅的耐久度。
    *   完美支持原版的 **“耐久” (Unbreaking)** 附魔算法，消耗几率为 `1 / (附魔等级 + 1)`。
*   **智能状态检查**:
    *   如果玩家在飞行途中脱下鞘翅或将其替换为其他装备（如胸甲），飞行模式会自动关闭。
    *   如果鞘翅在飞行过程中损坏，飞行模式会立即终止。
*   **权限系统**: 通过权限节点，精细控制哪些玩家可以使用此功能。
*   **本地化**: 提供完整的中文指令反馈和代码注释。

## 🎮 指令

*   `/fly` - 开启或关闭鞘翅飞行模式。
    *   **使用前提**: 必须穿戴鞘翅。

## 🔒 权限

*   `elytrafly.use` - 允许玩家使用 `/fly` 指令。
    *   **默认**: 仅服主 (OP) 拥有。

## 🚀 安装方法

1.  下载插件的 `.jar` 文件。
2.  将其放入你服务器的 `plugins` 文件夹内。
3.  重启服务器。

## 📋 环境要求

*   Spigot / Paper 1.21 或更高版本。
*   Java 21 (或与你服务器软件兼容的Java版本)。
